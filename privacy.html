Data Privacy

Privacy Policy

We believe you shouldn't have to sacrifice your data privacy to convert tester files. Here is exactly how your data is handled.

100% Local Processing

This application runs entirely in your web browser. When you select or drag-and-drop a Seaward PAT export file:

  • The file is read and processed in memory inside your browser.
  • No files, measurement fields, or test records are ever uploaded to a server or third party.
  • The conversion to CSV is executed locally on your device.

Cookies & Local Storage

This website does not use cookies or local storage tracking tokens. Your PAT file data is never stored in the browser beyond the current session.

Cloudflare Web Analytics

This website uses Cloudflare Web Analytics to collect aggregated, privacy-preserving page-view statistics (such as page visits, referrers, and country-level location). Cloudflare Web Analytics is designed to be cookieless and does not build individual visitor profiles or share data with advertising networks.

No PAT file content, measurement data, or personally identifiable information is included in these analytics reports. For full details on how Cloudflare handles this data, see the Cloudflare Privacy Policy.

Hosting & Infrastructure

The website's static files (HTML, CSS, JS, and image assets) are hosted using Cloudflare Pages. Cloudflare may collect standard server access logs (such as IP addresses and user agents) to prevent abuse and deliver content securely. This data is handled in accordance with Cloudflare's privacy practices and is not combined with your PAT data.

Seaward PAT Converter is an independent browser-based utility. All file parsing and conversion happen locally on your computer—no data is sent to a server.
