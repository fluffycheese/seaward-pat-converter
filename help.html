Converter Help

Non-technical guide

Export the tester file, then open it in your browser.

This page is written for someone who just wants the job done. It shows how to get the PAT export off the Apollo tester and how to use the converter on a normal web page.

No app install Works in a browser Download CSV for Excel
What you need

A browser, the Seaward export file, and access to this web page. You do not need PowerShell, Node.js, or anything installed on the PC.

Step 1

Get the file off the tester

1

Open the main menu on the Apollo tester

The manual shows the main menu and the key areas of the screen. The download option is reached from this menu.

2

Choose the download option that creates a file

Follow the download flow on the device and choose the option that saves the results to a file for PC use.

  • Use the USB-PC cable option if your setup uses a cable.
  • Use the flash memory stick option if your model supports it.
  • Save the export as a `.txt` file.
3

Transfer the `.txt` file to the computer

When the transfer finishes, copy or save the file somewhere easy to find, such as the Downloads folder.

Step 2

Use the converter

1

Open the converter page

Visit the converter in your browser. It is the same site as the help page, just the home page.

2

Choose the export file

Click Choose a PAT export and select the `.txt` file you saved from the tester.

3

Convert and download the CSV

Click Convert to CSV, then use Download CSV to save the spreadsheet file. Open that file in Excel if you want to sort or filter it.

Notes

Quick reminders

If the device screen wording is slightly different on your model, follow the option that creates a `.txt` export for use on a PC. The converter only needs the text file, not a special Seaward app.
