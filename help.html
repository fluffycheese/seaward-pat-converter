Seaward PAT Converter User guide
Converter Help

Non-technical guide

Export the tester file, then open it in your browser.

This page is written for someone who just wants the job done. It shows how to get the PAT export off the Apollo tester and how to use the converter on a normal web page.

No app install Works in a browser Download CSV for Excel
What you need

A browser, the Seaward export file, and access to this web page. You do not need PowerShell, Node.js, or anything installed on the PC.

Step 1

Get the file off the tester

1

Open the main menu on the Apollo tester

The manual shows the main menu and the key areas of the screen. The download option is reached from this menu using the arrow key or F5

2

Choose the download option that creates a file

Follow the download flow on the device and choose the option that saves the results to a file for PC use.

  • Select "To ASCII" on the "Download from Apollo" option
  • Use the USB-PC cable option
  • Download all using F4
3

Retrieving the file

The download is complete after the blue bar reaches 100%

  • The download appears as a new drive in Windows File Explorer ("TRANSFER E:" for example)
  • Open the drive and find the .txt file

    • Step 2

    Use the converter

    1

    Open this converter page

    Visit the converter in your browser. If you are reading this page, you are already there. Just select Converter in the top menu to go back to the main page.

    2

    Choose the export file

    Drag your exported .txt file into Choose a PAT export zone or click that zone to open a file picker.

    3

    Convert and download the CSV

    Click Convert to CSV

    • Wait for the conversion to finish
    • The converter will read the file and show a preview of the data.
    • Click Download CSV to save the spreadsheet file

    The CSV file can be opened in Excel or any spreadsheet program. It contains all the test results in a simple table format.

    Notes

    Quick reminders

    If the device screen wording is slightly different on your model, follow the option that creates a `.txt` export for use on a PC. The converter only needs the text file, not a special Seaward app.
